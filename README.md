# NDI Website

This repository now contains a clean Angular rebuild of the Noshahi Developers website.

## Structure

- `src/` – Angular application source
- `img/` – shared website assets
- `legacy/current-html/` – archived static HTML pages from the original site
- `legacy/` – earlier snapshot files kept for reference

## Scripts

- `npm install`
- `npm start`
- `npm run build`

## Notes

The Angular app uses standalone components, shared route content, and a common header/footer layout.
